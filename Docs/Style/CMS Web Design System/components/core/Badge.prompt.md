Pill status badge — color + text (never color alone) for program/report states.

```jsx
<Badge status="active" dot>Active</Badge>
<Badge status="warning" icon="pending">Review Required</Badge>
<Badge status="error" icon="warning">Action Required</Badge>
```

Statuses: `info`/`active` (sky), `success` (green), `warning` (gold), `error` (red), `neutral` (gray). Props: `icon` (Material Symbols), `dot`.
