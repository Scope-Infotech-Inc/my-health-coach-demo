WebGL/three.js rotating wireframe globe with glowing data points — national-reach hero accent. Loads three.js from CDN on demand; reduced-motion safe.

```jsx
<div style={{background:'var(--navy-deep)', borderRadius:'var(--radius-lg)'}}>
  <Globe3D height={320} points={32} />
</div>
```

Props: `height`, `points`, `spin`, `accent`, `globeColor`. Best on a navy surface.
