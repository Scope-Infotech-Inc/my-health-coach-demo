Inline alert / callout — tinted background, left accent rule, status icon.

```jsx
<Alert kind="info" title="Nondiscrimination in health">There have been recent court decisions affecting Section 1557…</Alert>
<Alert kind="error" title="Multiple failed login attempts" onClose={...} />
```

Kinds: `info` (sky), `success` (green), `warning` (gold), `error` (red). Props: `title`, `icon`, `onClose`.
