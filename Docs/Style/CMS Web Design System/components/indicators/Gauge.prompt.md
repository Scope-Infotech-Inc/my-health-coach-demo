Radial arc gauge (270°) with animated fill + count-up. For scores/utilization.

```jsx
<Gauge value={98.2} label="Compliance Rate" tone="navy" />
<Gauge value={42} max={60} unit="" label="Pending Reviews" tone="gold" />
```

Props: `value`, `max`, `size`, `label`, `unit`, `tone` (navy/gold/success), `thickness`.
